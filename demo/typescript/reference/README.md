# Reference artifacts (generated, for reference)

These two files are **generated by the Koine CLI** and committed here as a stable, browsable
snapshot — the live output tree (`demo/typescript/generated/`) is git-ignored and rebuilt on every
`run.sh`, so it never shows up in diffs. This folder gives reviewers something concrete to look at
without running the tool.

Both mirror the `Ordering` context in [`templates/starters/ordering`](../../../templates/starters/ordering).
Regenerate them (or the full tree) from the repo root:

```bash
dotnet run --project src/Koine.Cli -- build templates/starters/ordering --target typescript --out /tmp/ordering-ts
```

| File | What it is |
| --- | --- |
| `Order.ts.txt` | The `Ordering.Order` aggregate root — id, lines, status, entity equality |
| `OrderLine.ts.txt` | The `Ordering.OrderLine` value object — a derived `subtotal` getter (`unitPrice.multiply(quantity)`) |

The `.ts.txt` suffix is on purpose: the demo's own `tsc` run only ever type-checks
`demo/typescript/{src,generated}/**/*.ts`, so these reference copies are never picked up as
duplicate compilation units.

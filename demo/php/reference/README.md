# Reference artifacts (generated, for reference)

These two files are **generated by the Koine CLI** and committed here as a stable, browsable
snapshot — the live output tree (`demo/php/generated/`) is git-ignored and rebuilt on every
`run.sh`, so it never shows up in diffs. This folder gives reviewers something concrete to look at
without running the tool.

Both mirror the `Ordering` context in [`templates/starters/ordering`](../../../templates/starters/ordering).
Regenerate them (or the full tree) from the repo root:

```bash
dotnet run --project src/Koine.Cli -- build templates/starters/ordering --target php --out /tmp/ordering-php
```

| File | What it is |
| --- | --- |
| `Order.php.txt` | The `Ordering.Order` aggregate root — id, lines (a `list<OrderLine>`), status, entity equality |
| `OrderLine.php.txt` | The `Ordering.OrderLine` value object — a derived `subtotal()` method (`unitPrice->mul(quantity)`) |

The `.php.txt` suffix is on purpose: the demo's own `php -l` / `phpstan` run only ever checks
`demo/php/{generated,main.php}`, so these reference copies are never picked up as duplicate class
declarations.

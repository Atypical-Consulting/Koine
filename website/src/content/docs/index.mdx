## From ubiquitous language to working code

Koine is a compiler. You describe a bounded context in a small, readable DSL; it emits
self-contained, idiomatic C# — `sealed record` value objects with validating constructors,
entities with identity equality, aggregates with repositories, smart enums, domain events,
state machines, and more. The semantic model is **target-agnostic**, so the same `.koi` can
one day compile to TypeScript or Rust.

```koine
context Billing {
  enum Currency    { EUR, USD, GBP }
  enum OrderStatus { Draft, Submitted, Paid }

  value Money {
    amount:   Decimal
    currency: Currency
    invariant amount >= 0   "an amount cannot be negative"
  }

  aggregate Order root Order versioned {

    event OrderSubmitted {
      orderId:   OrderId
      lineCount: Int
    }

    value OrderLine {
      product:   ProductId
      quantity:  Int
      unitPrice: Money
    }

    entity Order identified by OrderId {
      customer: CustomerId
      lines:    List<OrderLine>
      status:   OrderStatus = Draft

      command submit {
        requires status == Draft   "only a draft order can be submitted"
        status -> Submitted
        emit OrderSubmitted(orderId: id, lineCount: lines.count)
      }
    }
  }
}
```

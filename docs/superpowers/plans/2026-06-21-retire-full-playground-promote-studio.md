# Retire the full Playground, promote Koine Studio — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Delete the redundant full Playground page, redirect its URLs into Koine Studio, and re-point the marketing site's hierarchy so Studio is the flagship and the inline mini playground is the first-contact taste.

**Architecture:** All changes live in `website/` (Astro Starlight site). `/playground/` becomes a tiny JS redirect stub that translates old URL shapes into Studio's `#model=` boot contract. The dual-mode `Playground` component collapses to a single embedded-only mode whose toolbar hands the live model off to Studio. Nav, hero, teaser, and docs links move from `/playground/` to `/studio/`. A new shared `encode.ts` provides the one canonical base64url encoder used by the controller (browser), the redirect, and the home teaser (build-time).

**Tech Stack:** Astro, TypeScript, CodeMirror (existing playground editor), the in-browser Koine WASM compiler. No test framework in `website/` — verification is grep + `npx tsc --noEmit` + a dev/build smoke pass, matching the site's established (test-free) convention.

## Global Constraints

- Site base path is `/Koine/` (`astro.config.mjs`); `import.meta.env.BASE_URL` is `/Koine/`. Build all internal hrefs as `${base}/…` where `const base = import.meta.env.BASE_URL.replace(/\/$/, '')`.
- Studio boots from `#model=<base64url-utf8>` only (`tooling/koine-studio/src/share.ts`). It does **not** read `?example=`.
- The base64url encoding is: UTF-8 bytes → base64 → `+`→`-`, `/`→`_`, strip trailing `=`. This must match across the controller, the redirect, and the teaser, hence the shared `encode.ts`.
- Do NOT modify Studio, the WASM bundle, the compiler, or `website/scripts/build-studio.mjs`.
- Commit identity (workspace rule): `git -c user.email=phmatray@gmail.com -c user.name="Philippe Matray" commit -m "…"`.
- Single-file sample ids available via `?example=`: `billing`, `ordering`, `contextmap`, `values` (generated by `scripts/build-samples.mjs` from `templates/starters/*`).

- [ ] **Step 1: Create the shared encoder**

Create `website/src/playground/encode.ts`:

```ts
// One canonical base64url(UTF-8) encoder, shared by the playground controller (browser),
// the /playground/ redirect, and the home-page teaser (build-time). Keeping a single copy
// means the Studio `#model=` handoff round-trips no matter which surface produced the link.
// btoa + TextEncoder are global in both modern Node (the Astro build) and the browser.
export function encodeCode(code: string): string {
  const bytes = new TextEncoder().encode(code);
  let bin = '';
  for (const b of bytes) bin += String.fromCharCode(b);
  return btoa(bin).replace(/\+/g, '-').replace(/\//g, '_').replace(/=+$/, '');
}
```

- [ ] **Step 2: Replace the full Playground page with a redirect stub**

Overwrite `website/src/pages/playground.astro` entirely with:

```astro
---
// /playground/ is retired in favour of Koine Studio. This page redirects every old Playground
// URL into Studio, preserving the model so shared links and bookmarks don't break:
//   #model=…       → /studio/#model=…              (real shared models — must round-trip)
//   ?code=<enc>    → /studio/#model=<enc>           (same base64url encoding)
//   ?example=<id>  → /studio/#model=<enc(sample)>   (old example deep links)
//   (none)         → /studio/
// Imports only the lightweight samples data + the shared encoder — never the CodeMirror controller.
import { SAMPLES } from '../playground/samples';
import { encodeCode } from '../playground/encode';

const base = import.meta.env.BASE_URL.replace(/\/$/, '');
const studio = `${base}/studio/`;
const exampleModels = Object.fromEntries(SAMPLES.map((s) => [s.id, encodeCode(s.code)]));
---

<!doctype html>
<html lang="en">
  <head>
    <meta charset="utf-8" />
    <title>Redirecting to Koine Studio…</title>
    <meta name="robots" content="noindex" />
    <link rel="canonical" href={studio} />
    <noscript><meta http-equiv="refresh" content={`0; url=${studio}`} /></noscript>
    <script define:vars={{ studio, exampleModels }}>
      (function () {
        var hash = location.hash.replace(/^#/, '');
        var params = new URLSearchParams(location.search);
        var dest = studio;
        if (/(^|&)model=/.test(hash)) {
          dest = studio + '#' + hash;
        } else {
          var code = params.get('code');
          var example = params.get('example');
          if (code) {
            dest = studio + '#model=' + code;
          } else if (example && exampleModels[example]) {
            dest = studio + '#model=' + exampleModels[example];
          }
        }
        location.replace(dest);
      })();
    </script>
  </head>
  <body>
    <p>Redirecting to <a href={studio}>Koine Studio</a>…</p>
  </body>
</html>
```
